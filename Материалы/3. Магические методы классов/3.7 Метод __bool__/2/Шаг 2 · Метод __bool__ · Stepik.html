Подвиг 1. Пусть в программе объявлен следующий класс:

class User:
    def __init__(self, name, old):
        self.name = name
        self.old = old

    def __len__(self):
        return self.old

И создаются два его экземпляра:

user1 = User('Sergey', 45)
user2 = User('Петр', 0)

Отметьте все верные пункты выполнения функции bool().
