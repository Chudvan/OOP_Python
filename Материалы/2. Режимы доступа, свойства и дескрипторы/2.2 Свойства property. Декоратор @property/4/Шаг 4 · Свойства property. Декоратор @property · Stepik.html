@JxSol, хм... мне при таком раскладе, PyCharm предупреждений не показывает

class Test:
    def __init__(self, x):
        self.x = x
        
    @staticmethod
    def check(x):
        return x > 0
        
    @property
    def x(self):
        return self.__x
    
    @x.setter
    def x(self, value):
        if self.check(value): self.__x = value
        else: self.__x = 1

ну и мне кажется более логичным, пользоваться механизмом проверки, который все равно уже реализован в сеттере, чем отдельно вызывать валидацию в __init__
