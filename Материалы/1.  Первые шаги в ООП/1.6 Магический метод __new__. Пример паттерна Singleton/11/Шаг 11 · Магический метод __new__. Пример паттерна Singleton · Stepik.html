Подвиг 10 (на повторение материала). В программе предполагается реализовать парсер (обработчик) строки (string) в определенный выходной формат. Для этого объявлен следующий класс:

class Loader:
    def parse_format(self, string, factory):
        seq = factory.build_sequence()
        for sub in string.split(","):
            item = factory.build_number(sub)
            seq.append(item)

        return seq

И предполагается его использовать следующим образом:

ld = Loader()
res = ld.parse_format("4, 5, -6.5", Factory())

На выходе (в переменной res) ожидается получить список из набора вещественных чисел. Например, для заданной строки, должно получиться:

[4.0, 5.0, -6.5]

Для реализации этой идеи необходимо вначале программы прописать класс Factory с двумя методами:

build_sequence(self) - для создания начального пустого списка (метод должен возвращать пустой список);
build_number(self, string) - для преобразования переданной в метод строки (string) в вещественное значение (метод должен возвращать полученное вещественное число).

Объявите класс с именем Factory, чтобы получать на выходе искомый результат.

P.S. В программе на экран ничего выводить не нужно.

Вместо фидбека напишу как можно было бы описать задание, в более понятном стиле.

Дан класс  Loader 

class Loader:
    def parse_format(self, string, factory):
        seq = factory.build_sequence()
        for sub in string.split(","):
            item = factory.build_number(sub)
            seq.append(item)

        return seq

Объект класса Loader парсит строку с разделителем "," возвращая в качестве результата список  в виде вещественных чисел.

Реализовать имплементацию класса Factory, с методами

  • build_sequence(self) - для создания начального пустого списка (метод должен возвращать пустой список);
  • build_number(self, string) - для преобразования переданной в метод строки (string) в вещественное значение (метод должен возвращать полученное вещественное число).

Пример:

ld = Loader()

res = ld.parse_format("4, 5, -6.5", Factory())

Результат : [4.0, 5.0, -6.5]

P.S. вероятно постановка задачи в таком стиле будет легче восприниматься.

Попробую немного прояснить условие :

Метод build_number(self, string) должен возвращать вещественное число, то есть float. Однако, float ( num)  можно применить только к тому, что isdigit(). И нельзя к строке "4, 5, 6, 7, 8.6". Значит - метод принимает всегда один единственный аргумент. В методе  build_number(self, string) сбивает с толку слово string - типа будет подаваться строка. Но это не так. Всегда будет подаваться то, что isdigit()

Давайте посмотрим на данную ниже функцию  def parse_format 

Сначала там в seq  присваивается метод возвращающий [ ]

Потом, в качестве итерируемого объекта нам как раз и предлагается взять строку  string, к которой применен сплит по запятым. - другими словами вот такой вот список [ 4, 5, 6, 7, 8.6 ]

И теперь для sub (это видимо сокращение от substring), нам уже предлагается использовать функцию  float(), которая у нас прописана в виде def build_number(...)

P.S.  Нетрудно догадаться, что если бы у нас на вводе оказалась вот такая строка " 3, 4, wow, [9,7], True,   , 5g, ;" -  то наша программа не смогла бы ее переварить.
