@Віталік_Коротких, создаете промежуточную переменную, например __adres. И еще одну переменную для счетчика. Каждый раз вызывая констуктор прибавляете счетчик и проверяете, если счетчик не переполнился то адресу назначете ссылку на объект супер... как на видео. И затем возвращается этот адрес. Ну раз уже поднялся этот вопрос - набросал небольшой пример, где показано, как модифицируются имена атрибутов с двойным подчеркиванием в имени при обращении к ним извне .  

class Test:
    __aaa = "это скрытый атрибут класса"
    aaa = "это открытый атрибут класса"

    def __init__(self):
        self.__bbb = "это скрытый атрибут экземпляра класса"
        self.bbb = "это открытый атрибут экземпляра класса"

    def test_access(self):
        # проверяем доступ ко всем атрибутам из метода класса...
        print(Test.aaa)
        print(Test.__aaa) # извне такие шутки не пройдут
        print(Test._Test__aaa)
        print(self.bbb)
        print(self.__bbb) # извне такие шутки не пройдут
        print(self._Test__bbb) 

t = Test()
t.test_access()

# Доступ извне ...
print(Test.aaa)
print(t.bbb)
#   так можно, но это плохо...
print(Test._Test__aaa)  # модификация имени для сокрытия
print(t._Test__bbb)     # модификация имени для сокрытия
