Подвиг 8. Объявите в программе класс Cart (корзина), объекты которого создаются командой:

cart = Cart()

Каждый объект класса Cart должен иметь локальное свойство goods - список объектов для покупки (объекты классов Table, TV, Notebook и Cup). Изначально этот список должен быть пустым.

В классе Cart объявить методы:

add(self, gd) - добавление в корзину товара, представленного объектом gd;
remove(self, indx) - удаление из корзины товара по индексу indx;
get_list(self) - получение из корзины товаров в виде списка из строк:

['<наименовние_1>: <цена_1>',
'<наименовние_2>: <цена_2>',
...
'<наименовние_N>: <цена_N>']

Объявите в программе следующие классы для описания товаров:

Table - столы;
TV - телевизоры;
Notebook - ноутбуки;
Cup - кружки.

Объекты этих классов должны создаваться командой:

gd = ИмяКласса(name, price)

Каждый объект классов товаров должен содержать локальные свойства:

name - наименование;
price - цена.

Создайте в программе объект cart класса Cart. Добавьте в него два телевизора (TV), один стол (Table), два ноутбука (Notebook) и одну кружку (Cup). Названия и цены придумайте сами. 

P.S. Отображать на экране ничего не нужно, только создать объекты по указанным требованиям.

Для таких как я:

1. Создайте класс, объявление объекта которого выглядит так: cart = Cart(). Локальный атрибут goods объекта  - пустой список.

2. Методы класса Cart следующие:

add(self, gd): Добавляет объект gd, ссылающийся на класс конкретного товара (создается далее) в список goods.

remove(self, indx): Удаляет объект из списка goods по индексу indx.

get_list(self) - создает окончательный список объектов товаров в форме: ['<наименовние_1>: <цена_1>',
'<наименовние_2>: <цена_2>', '<наименовние_N>: <цена_N>'] . Элементы списка берутся из локальных свойств объектов товаров. 

3. Объявите в программе следующие классы для описания товаров с локальными свойствами:

name - наименование; price - цена.

Table - столы;
TV - телевизоры;
Notebook - ноутбуки;
Cup - кружки.

объявление объектов которых выглядит так:

<произвольное имя> = ИмяКласса(name, price)

Создайте в программе объект cart класса Cart. Добавьте в него два телевизора (TV), один стол (Table), два ноутбука (Notebook) и одну кружку (Cup). Названия и цены придумайте сами. 

P.S. Отображать на экране ничего не нужно, только создать объекты по указанным требованиям.
