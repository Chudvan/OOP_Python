Подвиг 10. Объявите класс с именем Translator (для перевода с английского на русский) со следующими методами:

add(self, eng, rus) - для добавления новой связки английского и русского слова (если английское слово уже существует, то новое русское слово добавляется как синоним для перевода, например, go - идти, ходить, ехать);
remove(self, eng) - для удаления связки по указанному английскому слову;
translate(self, eng) - для перевода с английского на русский (метод должен возвращать список из русских слов, соответствующих переводу английского слова, даже если в списке всего одно слово).

Все добавления и удаления связок должны выполняться внутри каждого конкретного объекта класса Translator, т.е. связки хранить локально внутри экземпляров классов класса Translator.

Создайте экземпляр tr класса Translator и вызовите метод add для следующих связок:

tree - дерево
car - машина
car - автомобиль
leaf - лист
river - река
go - идти
go - ехать
go - ходить
milk - молоко

Сперва генерим пустой словарь:

d = {}

Потом набрасываем функцию с требуемым функционалом

def add(eng, rus):

   if eng not in d:  # Если передаваемого ключа нет в словаре (= if eng not in d.keys())

      d[eng] = [] # Добавляем такой ключ - значение = пустой список, а затем:

   d[eng].append(rus) # Аппендим этот пустой список, или уже не пустой, все равно надо.

Заработало? Потом вспоминаем, что экземпляр класса отличное именное пространство наподобие словаря. Для наших целей он и есть словарь. Осталось только пересмотреть первые лекции и вспомнить, как к нему обращаться.

class Translator:
    main_dict = {}

    def add(self, eng, rus):
        self.eng = eng
        self.rus = rus
        if eng in self.main_dict:
            k = self.main_dict.get(eng)
            k.append(rus)
            self.main_dict[eng] = k
        else:
            l = []
            l.append(rus)
            self.main_dict[eng] = l

    def remove(self, eng):
        del self.main_dict[eng]

    def translate(self, eng):
        return print(*self.main_dict.get(eng))
